Task:
Write down whether the following expressions return true, false, or raise an error.
Then, type the expressions into irb

(32 * 4) >= "129"
847 == '874'
'847' < '846'
'847' > '846'
'847' > '8478'
'847' < '8478'

Answers:
(32 * 4) >= "129" => raises an error
847 == '874' => false
'847' < '846' => false
'847' > '846' => true
'847' > '8478' => false
'847' < '8478' => false (wrong this evaluates to 'true')
